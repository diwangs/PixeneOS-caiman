# OTA Host

This branch hosts OTA for PixeneOS (GrapheneOS++)

This OTA comes in 2 flavors:

- [magisk](https://diwangs.github.io/PixeneOS-caiman/magisk/)
- [rootless](https://diwangs.github.io/PixeneOS-caiman/rootless/)

## Usage

- Choose the one that suits you the best
- Copy the URL of the flavor and set that in [Custota](https://github.com/chenxiaolong/Custota#usage)

## AVB Custom Key
The AVB public key metadata could be found in this [link](https://diwangs.github.io/PixeneOS-caiman/avb_pkmd.bin).
